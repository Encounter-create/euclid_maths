# No8. 母函数：数列的代数化身

> "A generating function is a clothesline on which we hang up a sequence of numbers for display."
> "生成函数是一根晾衣绳，我们把一串数字挂在上面向世人展示。" —— Herbert Wilf, *generatingfunctionology*

计数是组合数学的母题。从小学的枚举法，到中学的排列组合公式，我们一直在学着"数东西"。但有些计数问题极其刁钻：它们不直接问你 C(n,k)，而是藏在一个递推关系里，或者被分割成无数种情况。枚举？指数爆炸。分类讨论？无穷无尽。

面对这样的困局，数学家们发明了一件利器：**把数列装进一个函数里**。这个想法简单得近乎天真——把 a₀, a₁, a₂, ... 分别乘以 x⁰, x¹, x², ...，然后求和。于是一个无穷数列变成了一个函数：

$$F(x) = a_0 + a_1 x + a_2 x^2 + a_3 x^3 + \cdots$$

这就是**母函数**（Generating Function），也叫**生成函数**。你可能会问：把一个数列写成幂级数，有什么好处？好处是——代数工具可以上场了。

## 一、引入：多项式在计数什么

### 引入一：乘法口诀的隐藏含义

请你做一件小学就会做的事：展开 (1+x)²。

$$(1+x)^2 = 1 + 2x + x^2$$

这太简单了。但让我们换一个眼光来看：展开式中的系数 1、2、1，恰好是 C(2,0)、C(2,1)、C(2,2)。这不是巧合。

$$(1+x)^n = \binom{n}{0} + \binom{n}{1}x + \binom{n}{2}x^2 + \cdots + \binom{n}{n}x^n$$

展开式中 xᵏ 的系数，恰好是从 n 个元素中选 k 个的方案数。这就是二项式定理——人类最早的母函数。

让我们更进一步。如果我们要计算 Σ C(n,k)²，也就是"把 n 个球分成两堆，分别选 k 个的方案数之和"，怎么算？

观察：

$$(1+x)^n(1+x)^n = (1+x)^{2n}$$

左边展开后，xⁿ 的系数正是 Σ_{k=0}^{n} C(n,k)·C(n,n-k) = Σ C(n,k)²。右边展开后，xⁿ 的系数是 C(2n,n)。于是：

$$\sum_{k=0}^{n} \binom{n}{k}^2 = \binom{2n}{n}$$

我们只做了一次多项式乘法，就证明了一个并不显然的组合恒等式。代数替我们完成了组合推理。这就是母函数的第一重威力：**多项式乘法的卷积，恰好对应了组合计数的叠加**。

### 引入二：兔子问题的代数之解

公元 1202 年，斐波那契提出兔子繁殖问题。我们熟悉的递推：

$$F_0 = 0,\quad F_1 = 1,\quad F_{n+2} = F_{n+1} + F_n$$

现在，让我们定义斐波那契数列的母函数：

$$F(x) = F_0 + F_1 x + F_2 x^2 + F_3 x^3 + \cdots = x + x^2 + 2x^3 + 3x^4 + 5x^5 + \cdots$$

利用递推关系，我们可以玩一个代数魔术：

$$
\begin{aligned}
F(x) &= F_0 + F_1 x + \sum_{n \geq 2} F_n x^n \\
&= x + \sum_{n \geq 0} F_{n+2} x^{n+2} \\
&= x + \sum_{n \geq 0} (F_{n+1} + F_n) x^{n+2} \\
&= x + x\sum_{n \geq 0} F_{n+1} x^{n+1} + x^2\sum_{n \geq 0} F_n x^n \\
&= x + xF(x) + x^2 F(x)
\end{aligned}
$$

移项得：

$$F(x) - xF(x) - x^2 F(x) = x \quad\Rightarrow\quad F(x) = \frac{x}{1 - x - x^2}$$

我们得到了 F(x) 的**封闭形式**！接下来，用部分分式分解：

$$1 - x - x^2 = (1 - \phi x)(1 - \psi x)$$

其中 φ = (1+√5)/2，ψ = (1-√5)/2 是黄金比例及其共轭。展开为几何级数后提取系数，得到：

$$F_n = \frac{\phi^n - \psi^n}{\sqrt{5}}$$

这就是著名的 Binet 公式。我们用代数方法，把一个只能递推计算的数列，求出了闭式表达。

两个引入，两条核心线索：

- **多项式乘法 = 组合卷积**：把计数问题转化为代数运算
- **递推 + 母函数 = 封闭形式**：把无穷递推锁进一个方程，一步解出

## 二、基本技法

母函数方法看似神奇，其实核心操作只有三种。逐一掌握，便能上手。

### 1. 乘积便是卷积

这是母函数最基础的计数机制。

两个母函数 A(x) = Σ a_n xⁿ 和 B(x) = Σ b_n xⁿ 相乘：

$$A(x)B(x) = \sum_{n \geq 0}\left(\sum_{k=0}^{n} a_k b_{n-k}\right) x^n$$

xⁿ 的系数是一个**卷积**：a₀b_n + a₁b_{n-1} + ... + a_nb₀。

这个卷积恰好对应了"选 k 个做第一件事，选 n-k 个做第二件事"的计数逻辑。引入一中的 Σ C(n,k)² = C(2n,n) 就是这个道理。

更一般地：如果 A(x) 中 xᵏ 的系数是"第一类对象选 k 个的方案数"，B(x) 中 xʲ 的系数是"第二类对象选 j 个的方案数"，那么 A(x)B(x) 中 xⁿ 的系数就是"从两类对象中共选 n 个的方案数"。

**核心思想**：母函数的乘法模拟了独立选择的叠加。这是组合代数化的第一步。

**直观度评级**：★★★★★（直观、自然且强大）

### 2. 从递推到封闭形式

递推关系是组合计数的常见来源。第二种方法的思路是：只要能把递推翻译成母函数的代数方程，就有机会求出封闭形式。

一般步骤：

1. 设母函数 F(x) = Σ f_n xⁿ
2. 两边同乘适当的 x 的幂次，使递推下标对齐
3. 求和，得到关于 F(x) 的代数方程
4. 解出 F(x) 的封闭形式
5. 将 F(x) 展开为幂级数，读出系数 f_n（这需要系数提取技术）

**示例**：a₀ = 1，a_{n+1} = 2a_n + 1，求 a_n。

设 A(x) = Σ a_n xⁿ。两边同乘并对齐：

$$A(x) - a_0 = 2x A(x) + \frac{x}{1-x}$$

$$A(x) - 1 = 2x A(x) + \frac{x}{1-x}$$

$$A(x)(1-2x) = 1 + \frac{x}{1-x} = \frac{1}{1-x}$$

$$A(x) = \frac{1}{(1-x)(1-2x)} = \frac{2}{1-2x} - \frac{1}{1-x}$$

$$a_n = 2 \cdot 2^n - 1 = 2^{n+1} - 1$$

用母函数一气呵成，不需要"猜想→归纳"的忐忑。

**核心思想**：递推是局部的、无穷的；母函数把它变成了全局的、有限的代数方程。无穷被压缩进了有限。

**直观度评级**：★★★★（操作机械但需要部分分式技巧）

### 3. 把系数读回来

前面的操作是"从数列到函数"；这一步是"从函数回到数列"的逆向操作。常见技术：

**(a) 二项级数展开**

$$(1+x)^\alpha = \sum_{n \geq 0} \binom{\alpha}{n} x^n$$

其中 C(α,n) = α(α-1)...(α-n+1)/n! 适用于任意实数 α。

最常用的特例：

$$\frac{1}{(1-x)^{m+1}} = \sum_{n \geq 0} \binom{n+m}{m} x^n$$

**(b) 部分分式分解**

如果 F(x) 是有理函数（分子分母都是多项式），分解为若干形如 1/(1-ax)^k 的项之和，每一项都可以用 (a) 展开。

**(c) 乘积展开**

如果 F(x) = A(x)B(x)，用卷积公式提取系数。

**Catalan 数实例**：

Catalan 数 C_n 满足递推 C₀ = 1，C_{n+1} = Σ_{k=0}^{n} C_k C_{n-k}。

设 C(x) = Σ C_n xⁿ。递推的右边恰好是 C(x)² 的卷积形式：

$$\sum_{n \geq 0} C_{n+1} x^{n+1} = x \sum_{n \geq 0} \left(\sum_{k=0}^{n} C_k C_{n-k}\right) x^n = x C(x)^2$$

即 C(x) - 1 = x C(x)²。

解二次方程：

$$C(x) = \frac{1 - \sqrt{1 - 4x}}{2x}$$

用广义二项式定理展开 √(1-4x) 并提取系数，最终得到：

$$C_n = \frac{1}{n+1}\binom{2n}{n}$$

前几项：C₀=1, C₁=1, C₂=2, C₃=5, C₄=14, C₅=42。

**核心思想**：系数提取的本质，是把分析学工具（级数展开、部分分式、留数）用于组合计数。这是分析学与组合学最美丽的交汇之一。

**直观度评级**：★★★（技术性强，需要分析学基础）

## 三、实例剖析

方法就绪，来看实战。按照"识别信号 → 选择技法 → 执行计算"的节奏推进。

### 例题一 · 高考 / 基础层次

**问题**：证明组合恒等式 C(n,0)C(m,k) + C(n,1)C(m,k-1) + ... + C(n,k)C(m,0) = C(n+m, k)。

**识别信号**：求和指标在两个组合数之间滑动（第 k 个项是 C(n,k)C(m,k 的补)），这是**卷积结构**的典型信号 → 方法 1。

**解答**：左边恰好是 (1+x)^n 和 (1+x)^m 乘积中 x^k 的系数。

$$(1+x)^n(1+x)^m = (1+x)^{n+m}$$

右边展开，x^k 的系数为 C(n+m, k)。左边展开的 x^k 系数恰好是待证恒等式左边。等式成立。

**点评**：用组合论证（选 k 个人，从前 n 个男生和后 m 个女生中选）当然也可以，但母函数方法把推理变成了纯粹的多项式乘法——推理被代数化了。当你看到"两个组合数乘积求和"这类结构时，第一反应就应该是母函数。

### 例题二 · 高中竞赛层次

**问题**：把正整数 10 写成若干个正整数之和，有多少种方法？（不计顺序）

例如：10 = 3+3+2+1+1，10 = 5+3+2，10 = 10。这是经典的整数拆分（partition）问题，记 p(n) 为 n 的拆分数。

**识别信号**："若干个"意味着可以取任意多个，每个数可以重复使用 → 每个数的贡献是一个几何级数 → 方法 1 + 方法 2。

**解答**：考虑每个正整数 i 可以被选取 0, 1, 2, ... 次。选取 k 个 i 对应因子 x^{ik}。因此正整数 i 的贡献是：

$$1 + x^i + x^{2i} + x^{3i} + \cdots = \frac{1}{1 - x^i}$$

所有正整数合并：

$$P(x) = \sum_{n \geq 0} p(n) x^n = \prod_{i=1}^{\infty} \frac{1}{1 - x^i}$$

这就是 Euler 的整数拆分母函数。

对于 n=10，展开前几项（只需考虑 i≤10 的因子）：

$$\frac{1}{1-x} \cdot \frac{1}{1-x^2} \cdot \frac{1}{1-x^3} \cdots \frac{1}{1-x^{10}}$$

手工展开可得 p(10) = 42。

**点评**：整数拆分没有简单的闭式公式——p(n) 的渐近公式是 Hardy-Ramanujan 在 1918 年给出的，涉及 e^{π√(2n/3)}，是 20 世纪分析数论的巅峰杰作之一。但母函数给了我们一个优雅的编码方式：一个无穷乘积，精确描述了所有拆分可能性。无穷多种情况，被浓缩为一个函数——这就是母函数的编码力量。

### 例题三 · 更进一步

**问题**（Catalan）：在一个 n×n 的网格中，从 (0,0) 走到 (n,n)，每次只能向右或向上，且路径不能越过对角线 y=x。有多少种路径？

**识别信号**：路径计数、不可越过对角线、"n 的答案依赖小于 n 的答案" → 卷积递推 → 方法 2 + 方法 3。

**解答**：设 Catalan 数 C_n 为路径数。

考虑路径第一次接触对角线 y=x 的位置（除起点外）。假设在 (k,k) 处第一次回到对角线（1≤k≤n）。从 (0,0) 到 (k,k) 且不碰到中间对角线的子路径数为 C_{k-1}；从 (k,k) 到 (n,n) 为 C_{n-k}。

因此：

$$C_n = \sum_{k=1}^{n} C_{k-1} C_{n-k}$$

等价地：C_{n+1} = Σ_{k=0}^{n} C_k C_{n-k}，C₀ = 1。这正是方法 3 中处理的递推，最终结果：

$$C_n = \frac{1}{n+1}\binom{2n}{n}$$

**点评**：Catalan 数在组合数学中无处不在——合法括号序列、二叉树的形态数、凸多边形的三角剖分……同一个数列反复出现在看似毫无关系的问题中。母函数是发现这些等价性的统一语言：一旦两个问题导出相同的母函数方程，它们的答案就必然相同。这种"不同表象、同一本质"的发现，是数学中最令人兴奋的瞬间。

## 四、启发与一般化

### 什么时候用母函数？

四条核心识别信号：

| 信号 | 说明 | 对应方法 |
|------|------|----------|
| **卷积结构** | 求和形如 Σ a_k b_{n-k}，两个组合数乘积求和 | 方法 1 |
| **递推关系** | a_n 由前面项定义，尤其线性递推和非线性卷积递推 | 方法 2 |
| **"若干""任意个"** | 问题中存在"可以取 0 到任意多个"的计数逻辑 | 方法 1 + 2 |
| **间接计数** | 直接枚举爆炸，但"分阶段选择"的结构清晰 | 全部 |

### 如何联想到母函数？

**四步思维流程**：

1. **识别结构**：问题是"选 n 个东西"吗？如果是，设 xⁿ 的系数为答案。如果问题是关于一个数列 a_n，直接设 F(x) = Σ a_n xⁿ。
2. **建立方程**：看 a_n 的递推中是否有卷积（Σ a_k a_{n-k}）。有 → 乘积方程；没有但线性 → 线性方程；都没有 → 回到方法 1 找组合解释。
3. **求解封闭形式**：有理函数分解 / 二次方程 / 微分方程。这一步是技术活，但母函数方法的妙处在于：问题已经被转化为标准的代数/分析操作。
4. **提取系数**：用二项级数、部分分式或留数定理，读回 a_n 的闭式表达。

### 从 OGF 到更多

本文讨论的是**普通母函数**（Ordinary Generating Function, OGF）。当问题涉及**标号结构**（排列、集合划分、图计数），指数母函数（EGF）更为自然：

$$\hat{E}(x) = \sum_{n \geq 0} a_n \frac{x^n}{n!}$$

EGF 的乘法对应了"标号分配"的卷积，是处理排列类组合问题的利器。此外，Dirichlet 母函数在数论中大放异彩——Riemann zeta 函数就是其中最著名的例子。母函数的世界远比本文展示的更为辽阔。

### 母函数思想的哲学内涵

母函数方法最深刻的地方在于：它把"数东西"这个纯组合问题，转化为代数与分析问题。

你可以这样理解：当你在黑暗中摸索一个数列时，母函数就是一盏聚光灯。黑暗中你只能摸到前几项——1, 1, 2, 3, 5, 8...——聚光灯一打，你看到了 **1/(1-x-x²)** 这个完整的、优雅的表达。数列的"灵魂"被照亮了。

这就是数学中反复出现的一个主题：**升维**。把一个一维的数列嵌入到一个二维的对象（函数）中，高维空间中的操作（导数、积分、分解）映射回低维，就变成了深刻的组合结论。

学生时代我们学多项式，老师强调的永远是"化简"和"求值"；但母函数告诉我们：多项式的**形式**本身就有意义——系数在讲故事。

## 五、教学研究后记

母函数方法在中国中学数学课程中几乎是一个盲区。

课程中的排列组合，教的是公式、分类讨论、枚举。这些当然重要，但它们有一个共同的问题：面对稍复杂的问题（比如斐波那契递推、整数拆分、Catalan 数），这些初等方法立刻捉襟见肘。

母函数提供了一条完全不同的路径：把计数问题**代数化**。这不仅是解题技巧，更是一种世界观的转变——"数"与"式"在更高的层面上是统一的。

我在课堂上第一次讲母函数时，只讲了一个例子：(1+x)^n 的展开系数就是二项式系数。有学生问："这不就是二项式定理吗？有什么新鲜的？"我说："请你把二项式定理反过来读——不是'展开后系数是多少'，而是'什么样的数列可以被装进一个多项式里'。"

那个学生愣了一下，然后眼睛亮了起来。

这个"反过来读"的瞬间，就是数学思维真正的质变。从被动的"给我一个函数，我展开它"，到主动的"给我一个数列，我找一个函数来编码它"。前者是计算，后者是创造。没有一流的心性，就没有一流的技术——数学教会我们的，从来不只是"怎么算"，更是"怎么看"。

中学阶段引入母函数，不需要讲 EGF，不需要讲复分析和留数定理。只需要讲清楚一件事：多项式乘法对应计数的卷积，递推关系可以转化为母函数的代数方程。就这两点，已经足够打开一扇新的大门。

这也是"低门槛、高天花板"的好课题：入门只需要多项式乘法（初中水平），但它的纵深可以一直通到解析组合学（Analytic Combinatorics）——那个用复分析定量研究组合结构的前沿领域。每一个人，无论天赋高低，都可以从母函数中找到属于自己的收获：初学者学会用代数替代枚举，进阶者感受卷积结构的统一之美。

数学是一种信仰，热爱本身就是价值。希望这把母函数的火，能在你心中烧出一点光。
