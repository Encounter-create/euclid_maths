# No12. 无穷递降法：从不可能到必然的逻辑下降

## 1. 序言

> "I have discovered a truly marvelous proof of this proposition which this margin is too narrow to contain."
> —— Pierre de Fermat
> "我已发现此命题的一个绝妙证明，但此页边距太窄，无法容纳。"

1637 年，Fermat 在 Diophantus《算术》的页边写下这句著名批注时，他心中所想的证明方法，几乎可以确定就是**无穷递降法**（Method of Infinite Descent）。这是 Fermat 最钟爱的武器——在他幸存的书信中，他至少五次用它证明不同的命题，却从未正式发表过关于这个方法本身的任何系统论述。

那么，无穷递降法到底是什么？

让我们先看一个所有人都见过的证明——证明 $\sqrt{2}$ 不是有理数：

> 假设 $\sqrt{2} = a/b$，其中 $a, b$ 是互素的正整数。则 $a^2 = 2b^2$，故 $a$ 为偶数，设 $a = 2a_1$。代入得 $4a_1^2 = 2b^2 \implies b^2 = 2a_1^2$，故 $b$ 为偶数。但 $a, b$ 均为偶数，与"互素"矛盾。

这个证明几乎出现在每一本中学教材里。但很少有人意识到，它内部隐藏着一个比"互素矛盾"更深刻的逻辑结构：如果 $a/b = \sqrt{2}$，那么 $b_1 = a_1 = a/2$ 和 $a_1$ 满足 $a_1/b_1 = \sqrt{2}$ 且 $b_1 < b$。我们可以无限重复这个过程，得到一个严格递减的正整数序列 $b > b_1 > b_2 > \cdots$，但正整数不能无限变小——**矛盾**。

这个被大多数教科书跳过的"递降链"视角，正是无穷递降法的灵魂。本文要做的事，就是把这种被压缩在"显然矛盾"四个字里的深层逻辑，完整地展开、命名、分类，让它成为读者手中一把可以主动使用的武器。

---

## 2. 破题分析

### 良序原理：递降法的逻辑地基

无穷递降法的全部力量，来自一个看起来平凡到几乎不值得陈述的事实：

> **良序原理**（Well-Ordering Principle）：任何非空的正整数集合必有最小元。

这听起来像废话。但它有一个强大的逆否形式：**如果存在一个正整数的无限严格递减序列，那是不可能的**——因为如果有，这个序列本身构成了一个没有最小元的正整数集合。

无穷递降法的标准三段式由此而来：

1. **假设存在反例**：假设命题为假，取一个使某个正整数参数（如分母、解的大小等）**最小**的反例。
2. **构造更小反例**：从该最小反例出发，构造出另一个同类型的反例，其参数严格更小。
3. **矛盾**：这与"最小反例"的极小性矛盾，故假设不成立，原命题为真。

注意这里的一个微妙之处：**递降法本质上是归纳法在反证语境下的对偶形式。** 数学归纳法从基底出发，证明"如果 $P(k)$ 成立则 $P(k+1)$ 成立"，沿着自然数向上爬；无穷递降法从假设的反例出发，证明"如果存在 $P(n)$ 的反例，则存在 $P(m)$ 的反例且 $m < n$"，沿着自然数向下摔。两者是同一枚硬币的两面——我们将在启发与一般化中回到这个主题。

### 数学史注笔

无穷递降法的概念可以追溯到古希腊。Euclid 在《几何原本》第七卷中证明两数最大公约数存在时，使用的 Euclidean 算法暗含了递降思想——余数严格递减，最终终止于零。但将它系统化为一种证明"无解"类命题的方法，是 Fermat 的贡献。

Fermat 在 1659 年给 Carcavi 的信中，将无穷递降法描述为他最珍视的发明：

> "因为普通的归纳法无法处理无限的对象，我将归纳法与反证法结合，创造出一种特殊的证明方式，我称之为无穷递降。"

他用这个方法证明了 $x^4+y^4=z^4$ 无正整数解（$n=4$ 的 Fermat 大定理）、一个形如 $4k+3$ 的素数不能表示为两平方和、以及若干 Diophantus 问题的无解性。这些证明的共同特征令人着迷：**它们不构造任何东西，只证明某些东西不存在。** 无穷递降法在数学史上的独特地位，就在于它是第一个系统性的"不可能性"证明方法。

## 3. 解法详录

### 模式一：反证 + 递降 —— $\sqrt{2}$ 无理性的完整展开

让我们重写 $\sqrt{2}$ 的无理性证明，这一次用递降法的三段式框架完整展开，而不是把它压缩进"互素矛盾"四个字里。

**第一步：假设存在最小反例。**

假设存在正整数 $a, b$ 满足 $a/b = \sqrt{2}$。在所有这样的正整数对中，选取使 $b$ 最小的那一对——记作 $(a_0, b_0)$。注意，我们并不要求 $a_0, b_0$ 互素，因为递降法用极小性本身就蕴含了互素性（若它们有公因子，约分后得到更小的 $b$，与 $b_0$ 的最小性矛盾）。

**第二步：构造更小反例。**

由 $a_0^2 = 2b_0^2$ 知 $a_0$ 为偶数，设 $a_0 = 2a_1$。代入：

$$4a_1^2 = 2b_0^2 \implies b_0^2 = 2a_1^2$$

于是 $b_0$ 也是偶数，设 $b_0 = 2b_1$。现在：

$$\frac{a_0}{b_0} = \frac{2a_1}{2b_1} = \frac{a_1}{b_1} = \sqrt{2}$$

且 $b_1 = b_0/2 < b_0$。因此 $(a_1, b_1)$ 是一对满足条件但 $b$ 更小的正整数。

**第三步：矛盾。**

这与 $b_0$ 的最小性冲突。因此，不存在满足 $a/b = \sqrt{2}$ 的正整数对 $(a,b)$，即 $\sqrt{2}$ 不是有理数。

这个重构揭示了教科书证明中被遮蔽的一步：我们真正用到的不是 $a$ 和 $b$ "不能都是偶数"，而是 **"更小的 $b_1$ 仍然构成反例"**。前者只是后者在一步递降后的快照，而递降链的全部力量在于它可以无限重复——$b_0 > b_1 > b_2 > \cdots$，每一次都产生合法的反例，但正整数不能无限严格递减。良序原理才是那个按下扳机的最终力量。

---

### 模式二：Fermat 的镇山法宝 —— $x^4 + y^4 = z^4$

现在我们来面对一个严肃得多的问题：证明方程 $x^4 + y^4 = z^4$ 没有正整数解。这是 Fermat 大定理在 $n=4$ 时的情形，也是 Fermat 唯一留下完整证明的特例。

实际上，Fermat 证明了一个更强的命题：**$x^4 + y^4 = z^2$ 无正整数解。**（因为这蕴含了原命题——若 $x^4+y^4=z^4$ 有解，令 $w=z^2$ 即得 $x^4+y^4=w^2$ 的解。）

**第一步：假设存在最小反例。**

假设存在正整数 $(x,y,z)$ 满足 $x^4 + y^4 = z^2$。在所有解中选取使 $z$ 最小的那一个，记为 $(x_0, y_0, z_0)$。我们可以进一步假设 $\gcd(x_0, y_0) = 1$——因为如果有公因子 $d$，则 $(x_0/d, y_0/d, z_0/d^2)$ 也是解，且 $z$ 更小。

**第二步：构造更小反例。**

我们将 $(x_0^2, y_0^2, z_0)$ 视为勾股数组 $X^2 + Y^2 = Z^2$（其中 $X = x_0^2, Y = y_0^2, Z = z_0$）。由 $\gcd(x_0, y_0) = 1$ 知 $\gcd(X, Y) = 1$，因此 $(X, Y, Z)$ 是本原勾股数组。根据本原勾股数组的参数化公式，存在互素且不同奇偶的正整数 $m > n$，使得：

$$
\begin{aligned}
x_0^2 = X &= m^2 - n^2, \\[4pt]
y_0^2 = Y &= 2mn, \\[4pt]
z_0 = Z &= m^2 + n^2.
\end{aligned}
$$

（我们已经暗中排列使 $Y$ 为偶，$m,n$ 一奇一偶。）

现在观察第一式：$x_0^2 + n^2 = m^2$。因为 $m > n$ 且 $\gcd(m,n)=1$，这又是一个本原勾股数组。再次参数化：存在互素且不同奇偶的 $u > v$ 使得

$$n = 2uv,\quad m = u^2 + v^2,\quad x_0 = u^2 - v^2.$$

回到 $y_0^2 = 2mn$：代入 $m = u^2+v^2$ 和 $n = 2uv$ 得

$$y_0^2 = 2(u^2+v^2)(2uv) = 4uv(u^2+v^2).$$

因为 $\gcd(u, v) = 1$ 且 $u, v$ 不同奇偶，可知 $u, v, u^2+v^2$ 两两互素。而 $y_0^2$ 是平方数，$4$ 也是平方数，因此 $uv(u^2+v^2)$ 是平方数。两两互素意味着每个因子本身必须是平方数：

$$u = a^2,\quad v = b^2,\quad u^2 + v^2 = c^2.$$

最后一步：$u^2 + v^2 = c^2$ 即 $a^4 + b^4 = c^2$。于是 $(a, b, c)$ 是原方程的另一组解！

**第三步：矛盾。**

比较大小：

$$c \leq c^2 = u^2 + v^2 = m < m^2 + n^2 = z_0.$$

因此 $c < z_0$，与 $z_0$ 的最小性矛盾。故原方程 $x^4 + y^4 = z^2$ 无正整数解。

这个证明展示了一个重要的技术要点：**递降法在严肃数论证明中，构造更小反例往往不是一步完成的**——我们需要通过参数化、因式分解、互素性分析等多种工具"挖掘"出那个更小的解。但核心逻辑始终不变：存在最小反例 → 构造更小反例 → 矛盾。

### 模式三：递降到平凡基 case —— Vieta Jumping 回访

在上一期 No11 中，我们用 Vieta jumping 证明了 IMO 1988 第 6 题。现在回过头来审视那个证明的深层结构，会有一个令人惊讶的发现：**Vieta jumping 就是无穷递降法的一种特殊实现。**

回顾一下那个证明的核心步骤（以二次情形简化说明）：从 $(a, b)$ 出发，利用韦达定理得到另一根 $a' = kb - a$，且 $a' < a$。然后以 $(a', b)$ 为新的出发点，重复这个过程。最终下降链 $a > a' > a'' > \cdots$ 必须终止——终止的唯一可能是某一步产生 $a^{(n)} = 0$，此时推出原参数是完全平方数。

这里有一个关键的不同：

| | 模式一/二（经典递降） | 模式三（Vieta jumping） |
|---|---|---|
| **终点** | 逻辑矛盾 | 平凡可验证的基 case |
| **递降对象** | 反例的某个参数 | 解的某个分量 |
| **结论类型** | 某种对象不存在 | 某种对象必然具有某个性质 |

这不是递降法的"退化"，而是它的**变体**——递降链不一定非要摔死在矛盾的墙上，也可以安全着陆在一个可以直接验证的平凡基 case 上。选择哪种变体取决于命题的形式：

- 如果你要证明的是"不存在"，递降到矛盾。
- 如果你要证明的是"所有满足条件的东西都是 XXX"，递降到平凡 case 再回溯。

这个区分很实用：很多竞赛题用 Vieta jumping 时，学生容易迷失在"我到底在证明什么"——明确递降的终点类型，就能让证明逻辑重新清晰。

### 模式四：递降 + 回溯 = 完全分类 —— Markov 方程

让我们进一步提升递降法的地位：它不仅能证明"不存在"和"必然具有某性质"，还能**完整刻画一个解集合的全部结构**。最好的例子是 Markov 方程：

$$x^2 + y^2 + z^2 = 3xyz.$$

这个方程有无穷多组正整数解，包括 $(1,1,1)$、$(1,1,2)$、$(1,2,5)$、$(2,5,29)$ 等等。问题是：**如何确定一组给定正整数是不是解？如何生成所有解？**

答案藏在 Vieta jumping 里——但这次 jumping 不是为了下降，而是为了**生成**。

固定 $y$ 和 $z$，把 Markov 方程看作关于 $x$ 的二次方程：

$$x^2 - 3yz \, x + (y^2 + z^2) = 0.$$

设一个根为 $x$，由韦达定理，另一根为 $x' = 3yz - x$。注意 $x'$ 必然是正整数（因为 $3yz > x$ 恒成立，除非 $x$ 已经极小）。

现在，从任意一个解 $(x, y, z)$ 出发，固定其中两个分量不变，对第三个分量做 Vieta jumping。我们得到一个解树：

- 如果 $x > \max(y, z)$，则 $x' < x$——新解的"最大分量"变小了
- 重复这个操作，最大分量严格递减，直到无法继续——此时三个分量相等或出现某个平凡模式

可以证明：**所有解最终都递降到 $(1,1,1)$ 这一个根结点。** 反过来，从 $(1,1,1)$ 向上回溯所有可能的跳转，就生成了 Markov 方程的全部正整数解。

无穷递降法在这里扮演的角色从"否定武器"变成了"结构分析工具"——**递降建立唯一根，回溯生成全集合。** 这是递降法第三种、也是最高级的用法。

### 模式五：图论中的递降 —— 最小度贪心

递降法并非数论的专利。在离散数学中，有一种常见策略与递降法同构。

**定理**：任何有限无向图 $G$ 都有一个子图，其最小度至少为原图平均度的一半。

**第一步：识别递降对象。**

我们递降的不是"反例"（因为我们要证明存在性，而非不存在），而是**图本身**。考虑原图 $G$ 的平均度 $d$。如果 $G$ 中每个顶点的度数都 $\geq d/2$，那么 $G$ 本身已满足要求——无需递降。

**第二步：构造"更小"对象。**

否则，存在某个顶点 $v$ 度数 $< d/2$。从 $G$ 中删除 $v$ 及其关联边，得到 $G'$。关键性质：$G'$ 的平均度 $d'$ 满足 $d' \geq d$（删除低度顶点只会让平均度上升或不变）。

**第三步：终止与结论。**

图 $G$ 的顶点数严格递减（$|V| > |V'| > |V''| > \cdots$），而一个有限图不能无限失去顶点。最终我们停在一个所有顶点度数均 $\geq d/2$（此时平均度已升至至少 $d$ 水平）的子图。

这个证明展示了递降法脱离数论后的另一种形态：**递降对象可以是任何带有"有限 + 单调"结构的数学对象。** 在这里，"最小值"是顶点数，"递降步骤"是删除坏顶点，"单调不变量"是平均度不减。三要素齐全，递降法即可运转。

## 4. 启发与一般化

### 4.1 递降法工具箱：何时亮出这把刀

综合五种模式，我们可以提炼出四个信号——当你看到它们的任何一个时，就应该考虑递降法：

| 信号 | 典型措辞 | 递降对象 | 案例 |
|------|----------|---------|------|
| **命题否定** | "无解"、"不存在"、"不能" | 反例的某个正整数参数 | $\sqrt{2}$、$x^4+y^4=z^4$ |
| **自然数上界** | "正整数"、"最小"、"第一个" | 使某条件成立的最小正整数 | 模式一、二 |
| **解能生成更小解** | 韦达定理、递推关系 | 解的某个分量 | Vieta jumping、Markov |
| **去坏留好** | "删去不满足…的"、"贪心" | 对象规模（顶点数、区间长等） | 图论最小度 |

这四个信号的共性在于：它们都保证了你能够在一个有限结构中"往下走"，并且"走不远"——不是走到矛盾，就是走到一个可处理的终点。

实际使用时的决策流程：

1. 命题是否涉及"不存在"或"所有对象都必须满足 XXX"？
2. 如果是，能否将反例/对象与一个**严格单调递减的参数**关联？
3. 如果参数是自然数，良序原理自动保证递降的终止。
4. 如果参数不是自然数但有限（如图论中的点数、几何中的边长和），同样可以递降。

### 4.2 递降与归纳：一枚硬币的两面

让我们回到最根本的问题：无穷递降法和数学归纳法到底是什么关系？

**同一命题的两种证法。** 以 $\sqrt{2}$ 无理性为例——

**归纳法版本**：对所有正整数 $n$，命题 $P(n)$ = "不存在正整数 $a$ 使 $a/b = \sqrt{2}$ 且 $b = n$"。

- 基 case：$P(1)$ 显然成立（$a/1 = \sqrt{2}$ 意味着 $a = \sqrt{2}$，不是整数）。
- 归纳步：假设对所有 $k < n$，$P(k)$ 成立。若存在 $a$ 使 $a/n = \sqrt{2}$，则 $a^2 = 2n^2$，推得 $a$ 和 $n$ 都是偶数，于是 $(a/2)/(n/2) = \sqrt{2}$ 且 $n/2 < n$，与归纳假设 $P(n/2)$ 矛盾。

**递降法版本**：假设存在反例，取 $b$ 最小的反例 $(a_0, b_0)$。构造 $(a_0/2, b_0/2)$ 作为更小的反例，矛盾。

两个版本的逻辑骨架是完全一样的——"如果 $n$ 是反例，则 $n/2$ 也是反例"。区别仅在于叙事方向：归纳法说"小的情况成立，所以大的情况也成立"，递降法说"大的情况不成立，所以小的情况也不成立"。

两者的统一来自**良序原理**，它是归纳法和递降法的共同地基：

```
             良序原理
            /        \
    数学归纳法      无穷递降法
   (向上构造)      (向下销毁)
```

这一对偶关系有一个重要的教学含义：**如果一个学生已经理解了归纳法，他离理解递降法只差一层窗户纸**——把"假设 $P(k)$ 真"改成"假设 $P(k)$ 假"，把"推出 $P(k+1)$ 真"改成"推出存在 $j<k$ 使 $P(j)$ 假"。本质上，递降法就是归纳法在反证语境下的翻译。

## 5. 教学研究后记

在中学和大学低年级的数学教育中，无穷递降法通常被定位为"竞赛高级技巧"，在常规课程中几乎不出现。这是一个令人遗憾的错位。

首先，递降法在逻辑上并不比归纳法更难——如上所述，两者本质相同。把递降法推迟到竞赛阶段，等于把归纳法的一个自然延伸人为地神秘化了。$\sqrt{2}$ 的无理性证明出现在几乎每一本初中教材里，但几乎没有老师在这个点上停下来告诉学生"你们刚刚见证的方法叫无穷递降法，它的完整逻辑是这样的……"。这本该是递降法最天然的入门时刻。

其次，递降法与归纳法同步引入有一个独特的教学价值：**它让学生看到"证明"不是一个方向固定的活动。** 归纳法是从小往大走，递降法是从大往小走——两者的并存本身就是一个重要的数学思维训练：证明的方向是你选择的，不是命题强加的。

在数学史上，递降法也有一个有趣的"身份焦虑"。Fermat 反复使用它，却从未系统地命名和阐述它。他在信件中有时叫它"无穷递降"，有时只是描述操作步骤而不给名字。Euler 和 Lagrange 继承了这种方法，但同样没有给它写一本"方法论手册"。直到 19 世纪末，数论教材才开始将"无穷递降法"作为一个独立条目列入。一个被三位数学巨人频繁使用的武器，竟然花了两百多年才获得正式的"方法论地位"——这说明数学共同体对"方法"本身的反思，往往滞后于对"结果"的追求。
