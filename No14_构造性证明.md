# No14. 构造性证明：当"存在"不够用时

> "There are no non-experienced truths."
> — L.E.J. Brouwer, *On the Foundations of Mathematics* (1907)
>
> 「不存在未经心灵构造的真理。」

我们每天都在做证明。证明一个数是无理数、证明一个不等式恒成立、证明某个方程有解。这些证明有一个共同的潜台词：我们证明了**存在**。

但存在和存在不一样。有一种证明只告诉你"一定有"，却不说"在哪里"。另一种证明挽起袖子，把答案摆在桌面上。

---

## 1. 破题分析：两行证明中的幽灵

让我们从一个只有两行的著名证明开始。

**命题**：存在两个无理数 $a, b$，使得 $a^b$ 是有理数。

**证明**：考虑 $\sqrt{2}^{\sqrt{2}}$。
- 若它是有理数：取 $a = b = \sqrt{2}$，得证。
- 若它是无理数：取 $a = \sqrt{2}^{\sqrt{2}}$，$b = \sqrt{2}$，则

$$a^b = \left(\sqrt{2}^{\sqrt{2}}\right)^{\sqrt{2}} = \sqrt{2}^{2} = 2$$

是有理数，得证。

两行，干净利落。从逻辑上，这个证明无懈可击。它告诉我们：**一定存在**这样一对无理数。

但如果你把这两行证明交给一位程序员，让她写一段代码输出这对数，她会完全无从下手——因为证明从头到尾没有告诉我们 $\sqrt{2}^{\sqrt{2}}$ 到底是不是有理数。如果我问你"到底是哪种情况"，你只能说：我不知道，但至少有一种情况成立，这就够了。

这就是**非构造性证明**和**构造性证明**的核心差异。一个告诉你门后面有宝藏；另一个把钥匙交到你手里。

构造性证明的行为准则是：**不假设排中律**——你不能说"要么 P 要么非 P"，除非你能判定到底是哪一个。你只能从已知的、已经构造出的对象出发，一步步搭建出结论中的对象。

---

## 2. 经典案例：四种构造策略

构造不是"灵光一闪"的艺术。经过几代数学家的实践，构造性证明已经沉淀为一套工具箱。下面我们系统讲解四种最核心的构造策略。

### 2.1 贪心构造

**核心思想**：每一步做当前看起来最好的选择，最终拼出全局最优解。

贪心构造的威力在于：当你面对的选择有"局部最优即全局最优"的结构时，你只需要在每个岔路口做那个最直接的决定，路自然会走到终点。

**经典例子：区间调度问题**

给定 $n$ 个区间 $[l_i, r_i)$，要求选出尽可能多的两两不相交的区间。

构造策略很简单：按**结束时间**从早到晚排序，依次选取。每选一个区间后，跳过所有与它相交的后续区间。

等等——这听起来太简单了，凭什么保证它是最优的？

**证明（交换论证）**：设贪心选出的区间集合为 $G = \{g_1, g_2, \ldots, g_k\}$，按结束时间排序。任取一个最优解 $O = \{o_1, o_2, \ldots, o_m\}$ 也按结束时间排序。归纳证明：对每个 $i$，$g_i$ 的结束时间不晚于 $o_i$ 的结束时间——因为在第 $i$ 步，贪心选了可选中最早结束的区间，而 $o_i$ 在可选范围内，所以 $g_i$ 不可能比 $o_i$ 结束得更晚。由此推出 $k \ge m$，而 $m$ 是最优解的大小，故 $k=m$。

整个构造过程中，没有任何一步需要"猜"——排序 → 遍历 → 贪心选 → 证最优。每一步都是机械可执行的。

**识别信号**：当你面对的问题可以在每一步独立决策，且"尽早结束不选的选项"不会妨碍后续选择时，贪心构造是你的第一候选。

### 2.2 归纳构造

**核心思想**：从基例出发，每一步用归纳假设添加一个新元素，逐层搭建。

归纳法大家从高一开始就学了——证明某个性质 $P(n)$ 对所有正整数成立。但在构造性证明中，归纳法的角色不只是"证明 $P(n)$ 真"，更是"从 $P(n-1)$ 的已构造对象出发，构造出 $P(n)$ 的对象"。

**经典例子：生成树的归纳构造**

命题：任何 $n$ 阶连通图都包含一棵生成树。

对 $n$ 归纳。基例 $n=1$：单点图本身就是一棵生成树。归纳步：假设所有 $k$ 阶连通图都含生成树（$k < n$）。任取 $n$ 阶连通图 $G$ 的一个顶点 $v$，删除它得到 $G' = G - v$。$G'$ 可能不连通，设其连通分量为 $C_1, C_2, \ldots, C_t$。每个 $C_i$ 是阶数小于 $n$ 的连通图，由归纳假设各含一棵生成树 $T_i$。由于 $G$ 连通，$v$ 与每个 $C_i$ 至少有一条边相连——自每个 $C_i$ 各取一条这样的边 $(v, u_i)$。将这些边连同各 $T_i$ 拼在一起，即得 $G$ 的一棵生成树。

这个证明不仅告诉你"存在生成树"，而且给了你一个构造它的递归算法：删点 → 对每个连通分量递归构造 → 用删去的点连回来。你可以在任何连通图上实际运行这个过程。

**识别信号**：当对象按自然数索引，且第 $n$ 个对象可以由第 $n-1$ 个对象"加一点东西"得到时，归纳构造是最自然的武器。

### 2.3 代数构造

**核心思想**：用代数运算或参数化直接写出对象——不靠归纳、不靠贪心，而是靠符号计算把答案"算"出来。

**经典例子：勾股三元组的参数化**

我们从小就知道 $3^2 + 4^2 = 5^2$，也记得 $5^2 + 12^2 = 13^2$。但无穷多组勾股三元组的**存在**本身并不有趣——真正有趣的是：**有没有一个公式，把全部勾股三元组一次性生成出来？**

答案是有的。取任意两个正整数 $m > n$，令

$$a = m^2 - n^2, \quad b = 2mn, \quad c = m^2 + n^2$$

则 $a^2 + b^2 = (m^2 - n^2)^2 + (2mn)^2 = m^4 - 2m^2n^2 + n^4 + 4m^2n^2 = (m^2 + n^2)^2 = c^2$。反过来，任何本原勾股三元组（$a,b,c$ 互素）都可以写成这种形式。因此这三个代数表达式是勾股三元组问题的**完全解**——不仅证明了有无穷多组，而且把每一组都握在了手里。

代数构造的核心洞察是：很多组合和数论问题，其背后的对象本身由代数关系定义。一旦你找到了这些关系在代数层面的"可逆"表达，构造就变成了代入参数。

**识别信号**：当问题的条件可以用多项式方程或矩阵方程表述，且你怀疑解有参数化形式时，代数构造往往能一击致命。

### 2.4 随机构造（概率方法）

**核心思想**：不直接造出具体的对象，而是证明**随机选取一个对象，以正概率满足条件**——因此满足条件的对象一定存在。

这种方法听起来像在耍赖：你没有告诉我那个对象是什么，你只是说"随便选一个，有很大可能就对了"。但它确实是一种构造性证明——它不假设排中律，它实实在在地证明了存在，只是没有给出确定性的选取算法。

**经典例子：Erdős 对 Ramsey 数下界的证明**

Ramsey 数 $R(k,k)$ 是最小的 $n$，使得任何 $n$ 个顶点的完全图的边二染色（红/蓝）都必然包含一个单色的 $k$ 阶完全子图。

我们想证明 $R(k, k) > 2^{k/2}$——也就是说，存在一个顶点数 $n < 2^{k/2}$ 的染色方案，其中没有任何单色的 $K_k$。

构造方法：**随机染色**。对 $n$ 个顶点的完全图的每条边，独立地以 $\frac{1}{2}$ 概率染红色、$\frac{1}{2}$ 概率染蓝色。固定任意 $k$ 个顶点——它们构成一个 $K_k$，这个 $K_k$ 全部是红色的概率为 $2^{-\binom{k}{2}}$，全部是蓝色的概率也是 $2^{-\binom{k}{2}}$。所以这组 $k$ 个顶点形成单色 $K_k$ 的概率为 $2 \cdot 2^{-\binom{k}{2}}$。

顶点中 $k$ 元子集共有 $\binom{n}{k}$ 个。由联合界不等式：

$$P(\text{存在单色 } K_k) \le \binom{n}{k} \cdot 2 \cdot 2^{-\binom{k}{2}}$$

当 $n < 2^{k/2}$ 且 $k$ 足够大时，可以证明这个概率严格小于 1。因此至少存在一种染色方案不含单色 $K_k$。

我们确实没有给出具体的染色方案。但每个步骤都是可验证的：随机染色的概率空间是明确定义的，联合界的计算是严格的，概率 < 1 意味着正概率事件非空——存在性由此确立。Erdős 有一句名言描述这种方法：「如果你想证明某个对象存在，构造一个随机空间，然后证明它以正概率出现。」

**识别信号**：「存在一个……使得……」+「想要紧的下界/上界」+ 直接构造过于复杂——概率方法往往是最优雅的选择。

## 3. 动手试试：三道题，三个视角

理论讲完了。下面我们用三道题检验你是否真的掌握了构造性思维——不需要竞赛功底，只需要对数学有好奇心。每道题的讲解按照"读题 → 破题 → 求解 → 复盘"四个步骤展开。

### 2.1 归纳构造：二进制表示

**题目**：证明任何一个正整数 $n$ 都可以表示为若干个**互不相同**的 $2$ 的幂之和。例如 $13 = 8 + 4 + 1 = 2^3 + 2^2 + 2^0$。

你已经知道答案了——这就是二进制。但我们能不能不依赖"二进制"这个先入为主的概念，纯粹用归纳法构造出来？

**破题**：归纳法天生适合处理「正整数」——基例 $n=1$ 显然（$1=2^0$）。归纳步：假设所有小于 $n$ 的正整数都已经能用不同 $2$ 的幂之和表示了，我们要表示 $n$。

关键想法：找一个不超过 $n$ 的最大的 $2$ 的幂，记作 $2^k$，然后将剩下的部分 $r = n - 2^k$ 交给归纳假设——因为 $r < 2^k$（否则 $2^{k+1} \le n$，与"最大"矛盾），且 $r < n$，归纳假设适用。

**求解**：对于 $n=1$，$1 = 2^0$，完成。假设对所有小于 $n$ 的正整数结论成立。取 $k$ 使 $2^k \le n < 2^{k+1}$。令 $r = n - 2^k$。若 $r = 0$，则 $n=2^k$，完成。若 $r > 0$，则 $r < n$，由归纳假设 $r$ 可表示为互不相同的 $2$ 的幂之和。又因为 $r < 2^k$，这些幂的指数全都小于 $k$——因此加上 $2^k$ 后，所有项仍然互不相同。于是 $n = 2^k + \text{(r的表示)}$ 即为所求。

**复盘**：如果你觉得"这不就是二进制转换"，你说得对——这正是二进制存在的**构造性证明**。归纳法不仅回答了「每个数都能写成二进制」，而且给了你一个具体算法：每次减去不超过当前数的最大 $2$ 的幂，直到归零。你可以现在就用 37 试一下：$37 \to 37-32=5 \to 5-4=1 \to 1-1=0$，所以 $37 = 32+4+1$。归纳构造就藏在你每天用的算法里。

### 2.2 直接构造：走出幽灵证明的阴影

**题目**：找一个**确定**的无理数 $a$ 和一个**确定**的无理数 $b$，使得 $a^b$ 是有理数。不准用分情况讨论——你必须同时给出 $a$ 和 $b$ 的具体值，并验证 $\sqrt{2}^{\sqrt{2}}$ 那种「两种case总有一个成立」的滑头不算数。

这道题的来源正是 §1 中的幽灵证明——那个证明只告诉你「存在这样一对」，却不说到底是哪一对。现在我们要求：**写出具体的 $a$ 和 $b$，让任何人拿计算器都能验证。**

**破题**：我们需要两个确定的、已知是无理数的数，其中一个作为底数、一个作为指数，使得结果是一个简洁的有理数。对数是一个天然的方向：如果 $a$ 是无理数，而 $b = \log_a q$（其中 $q$ 是有理数），那么 $a^b = q$ 自动是有理数。问题转化为：能不能选一个无理数 $a$ 和一个有理数 $q$，使得 $\log_a q$ 也是无理数？

**求解**：取 $a = \sqrt{2}$（已知无理数），$q = 3$（有理数）。令 $b = \log_{\sqrt{2}} 3$。那么

$$a^b = \sqrt{2}^{\log_{\sqrt{2}} 3} = 3$$

现在验证 $b$ 是否为无理数。若 $\log_{\sqrt{2}} 3 = \frac{p}{q}$（$p,q$ 互素），则

$$(\sqrt{2})^{p/q} = 3 \quad\Rightarrow\quad 2^{p/(2q)} = 3 \quad\Rightarrow\quad 2^p = 3^{2q}$$

左边是 $2$ 的幂，右边是 $3$ 的幂。唯一使等式成立的整数解是 $p=q=0$（整数唯一分解定理），矛盾。因此 $b$ 是无理数。

于是我们找到了确定的一对：$a = \sqrt{2}$，$b = \log_{\sqrt{2}} 3$。两人都是无理数，$a^b = 3$ 是有理数。任何人不信都可以拿计算器验：$\sqrt{2} \approx 1.414$，$\log_{\sqrt{2}}3 \approx 3.170$，$1.414^{3.170} \approx 3.000$。

**复盘**：这道题的精髓在于——**用代数构造绕开了排中律**。幽灵证明说「要么A要么非A，无论哪种结论都成立」。我们不理它，直接找了第三条路：自己造一对出来。这就是构造性证明的信条：与其争论一个你不知道答案的问题，不如找一个你知道答案的问题来做。

### 2.3 代数构造：从一个解生出无穷个

**题目**：观察等式 $3^2 + 4^2 = 5^2$。不用一个一个猜，直接**写出一个公式**，生成无穷多组满足 $x^2 + y^2 = z^2$ 的正整数解 $(x, y, z)$。

铺垫到这里，你应该立刻反应过来——这就是 §2.3 的勾股三元组参数化。但我们要做的不是照搬结论，而是**自己把公式推出来**。

**破题**：$x^2 + y^2 = z^2$ 可以改写为 $(x/z)^2 + (y/z)^2 = 1$。这就是说，$X = x/z$，$Y = y/z$ 是单位圆上的有理点。而单位圆上的有理点可以通过**一条过已知有理点的直线**截出——方程是二次的，直线与圆的第二个交点必然也是有理点（因为二次方程的一个根已知是有理数，另一个根由韦达定理可求，自动是有理数）。

**求解**：取已知有理点 $P(-1, 0)$。过 $P$ 作斜率为 $t$ 的直线（$t$ 为任意有理数）：$Y = t(X+1)$。代入单位圆 $X^2+Y^2=1$：

$$X^2 + t^2(X+1)^2 = 1$$
$$(1+t^2)X^2 + 2t^2 X + (t^2 - 1) = 0$$

已知一个根 $X=-1$。设另一个根为 $X$，由韦达定理：

$$-1 \cdot X = \frac{t^2-1}{1+t^2} \quad\Rightarrow\quad X = \frac{1-t^2}{1+t^2}$$

进而 $Y = t(X+1) = t\left(\frac{1-t^2}{1+t^2}+1\right) = \frac{2t}{1+t^2}$。

令 $t = n/m$（$m, n$ 为任意正整数，$m > n$ 以确保 $X>0$），通分得

$$X = \frac{m^2-n^2}{m^2+n^2}, \quad Y = \frac{2mn}{m^2+n^2}$$

回代 $X = x/z$，$Y = y/z$，取 $z = m^2+n^2$：

$$x = m^2 - n^2, \quad y = 2mn, \quad z = m^2 + n^2$$

代入 $m=2, n=1$：$x=3, y=4, z=5$——熟悉的 $3^2+4^2=5^2$。$m=3, n=2$：$x=5, y=12, z=13$——$5^2+12^2=13^2$。$m=4, n=1$：$x=15, y=8, z=17$——$15^2+8^2=17^2$。每给一组 $(m,n)$，就生成一组解，无穷无尽。

**复盘**：这条推导路的每一步都是可操作的——画圆、过定点作直线、解二次方程、用韦达求另一根、有理化参数。没有一步依赖「假设存在」，每一步都是「写下来」。代数构造的核心就在这种操作感中：**你不需要先相信解存在，你只需要把解算出来。**

## 4. 启发与一般化

### 2.1 四种构造策略的统一基因

表面上看，贪心构造是"局部决策"，归纳构造是"逐层堆叠"，代数构造是"符号运算"，随机构造是"概率论证"——它们的操作对象和手法完全不同。但它们的底层共享着同一个信念：**存在性断言必须被翻译为可执行的步骤**。

从这个角度看，四种策略不过是这同一个信念在不同数学语境下的四种方言。当你面对一个"证存在"的问题时，第一步不是去翻工具箱里的具体技巧，而是先转换问题的表述方式——把"存在 $x$ 使得 $P(x)$"改写为"如何找到 $x$ 使得 $P(x)$？"这个改写动作本身，就是构造性思维的入门仪式。

### 2.2 方法选择指南

| 问题特征 | 首选策略 | 判断关键词 |
|---------|:------:|----------|
| 每一步的局部最优选择不互相干扰 | 贪心构造 | "最早/最短/最小"+"不冲突" |
| 对象按自然数索引，能从 $n-1$ 推到 $n$ | 归纳构造 | "对 $n$ 归纳"+"添加一个元素" |
| 条件可由多项式/矩阵等式表述 | 代数构造 | "参数化""通解""生成函数" |
| 存在性/下界，直接构造过于复杂 | 随机构造 | "存在"+"概率 > 0"+"期望" |

### 2.3 什么时候该构造，什么时候不必

并非每道"证存在"的题都适合用构造法重写。有些"存在"过于抽象，构造的代价远超问题的价值。判断标准：

- **对象的可表示性**：构造出的对象能否用有限信息描述？（二进制表示 → 每步减一个 $2^k$，√；$\log_{\sqrt{2}}3$ → 确定的无理数表达式，√；Ramsey 图 → 边染色方案需要指数级描述，但理论上有限，√；选择公理下的 Hamel 基 → 不可描述，✗）
- **构造的成本**：构造算法是否在可接受的时间内完成？（贪心分区 → $O(n\log n)$，√；勾股三元组 → 每代入一对 $(m,n)$ 即生成一组，√；随机染色 → 需要指数级验证，但在理论上算构造，半√）
- **构造的附加价值**：除了"证存在"，构造还额外提供了什么？（归纳构造 → 同时给出算法；代数构造 → 给出通解公式；鸽巢原理的构造化 → 找出具体哪两个数）

回看 No1 到 No13，构造性思维其实早已悄然穿梭其间。

No1 鸽巢原理的证明看似是"如果每个箱子最多一只鸽子，总数不超过箱子数"的否证推理，但它实质上指向一个构造性问题：**哪只**鸽子是多余的？No3 网格作图——整篇文章就是一次漫长的构造练习，从两个格点出发，用无刻度直尺画出中点、平行线、垂线。No8 母函数——为数列构造一个生成函数，代数的形式化操作反过来为组合问题提供了解。No11 韦达定理——从两条根上的代数关系构造出系数上的对称多项式。No12 无穷递降法——每一步构造一个更小的反例，无限下降本身就是一种构造性存在断言（"如果存在反例，则存在最小反例；构造更小的反例，矛盾"）。No13 群作用——Burnside 引理的核心是**数不动点**，而不动点的识别本身就是一个构造问题：如何判定某个染色方案在旋转下不变？

如果把 Nos 1-13 比作一座建筑，那么构造性证明不是某面墙上的装饰，而是贯穿地基到屋脊的钢筋。

## 6. 数学史插曲：Hilbert 对阵 Brouwer

1920 年代的数学界爆发了一场哲学大战。对阵双方是当时最顶尖的两位数学家。

David Hilbert 认为，存在性证明是数学的精髓。只要你能逻辑一致地推出"存在这样一个数"，就不需要告诉我它具体是什么。他有一句著名的话：「禁止数学家使用排中律，就像禁止天文学家使用望远镜、禁止拳击手使用拳头。」

Luitzen Brouwer 激烈反对。他认为数学根植于人类心灵的**构造**——你只有把一个对象实实在在地构造出来，才有资格说它存在。他说：「你不能说你有一个数，却不告诉我它是什么。那就像说你有一个朋友，却从没见过他。」

这场论战持续了十余年，最终 Hilbert 一派在主流数学中获胜——我们今天使用的绝大多数数学证明，都允许非构造性的推理。但 Brouwer 的幽灵从未散去。

1967 年，美国数学家 Errett Bishop 写了一本书——《构造性分析基础》。他在书中证明了一个惊人的事实：几乎全部经典分析都可以用严格的构造性方法重写，而且不会丢失任何实质内容。Bishop 不是 Brouwer 的门徒——他不否定经典数学的有效性，他只是证明：**构造性不是一种限制，而是一种选择。当你选择构造时，你获得的不只是一个"是"或"否"，而是一套可执行的方案。**

本文的主题，正是这个选择。
