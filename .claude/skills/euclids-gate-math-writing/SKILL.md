# 欧几里德之门 数学公众号写作

> **配套交互规则**：写作时必须同时加载 `euclids-gate-writing-interaction` skill，遵守三条交互原则——不确定即追问、多路径必枚举、先提纲后正文。

## 精神纲领

写作时始终铭记以下六条核心思想，它们决定了文章的基调和价值取向：

### 1. 人是终极目的

科学技术只是手段，人的本身才是终极目的。我们希望以数学启发人的心灵，以几何强大人的思维。只有心的强大，才是真正的强大，只有精神的富足，才是真正的幸福。通过数学教育，可以实现人的幸福。每一篇文章的最终落点，都应该是人的成长，而非知识的堆砌。

### 2. 教育本质：点燃火焰

教育不是灌输，而是点燃火焰（苏格拉底）。教育不是把篮子装满，而是点亮一盏灯，唤醒一个沉睡的灵魂（雅斯贝尔斯）。每一篇文章都应致力于唤醒读者的思考，而非单纯传递知识。好的数学教育，让一棵树摇动另一棵树，一朵云推动另一朵云，一个灵魂唤醒另一个灵魂。

### 3. 数学美学

随机中的确定，混沌中的有序，偶然中的必然。追求优雅、精彩、和谐的数学，在文章中展现数学的结构之美。数学，正当的理解，不仅拥有真理，还拥有至高无上的美——一种如雕塑般冷峻而朴素的美（罗素）。

### 4. 热爱即天赋

天赋自带比较的滤镜，不应将每个数学学习者与拉马努金比较。对于真正热爱数学的人，热爱本身就是价值，热爱本身就是最珍惜的天赋。数学是一种信仰，热爱本身就是价值。文章应该鼓励读者找到自己的热爱，而非制造焦虑。猴子有猴子的森林，鱼有鱼的大海——每个人都有自己的天赋和热爱。

### 5. 数学即心性修炼

没有一流的心性，就没有一流的技术。数学学习的过程，本质是一个锻炼心性的过程。数学教会我们专注、思考、判断：什么是美的，什么是优雅的，什么是重要的，什么是次要的。一个不专注不纯粹的人，不可能具有出色的注意力和科学 taste。身边数学好的朋友，无一不是专注而纯粹的。文章应在传授知识的同时，传递这种心性修炼的价值——数学让我们更好地 navigate through life，成为一个更好的人。

### 6. 社会关怀与师者修养

关注社会现实，观察应试教育下个体的发展困境，助力资源劣势学生打破信息不对称，对抗教育资本化。文章应让优质数学思维触手可及。教书匠的自我修养，是从"术"到"道"的升华——做题只是手段，学习才是目的。做一个真正的灵魂工程师。

## 写作宗旨：何以所以然

所有文章的根本目标是让学生理解"何以所以然"——这个概念为什么长这样？这个方法为什么有效？数学家当初是怎么想到的？为什么这么做而不是那么做？

这决定了文章的结构选择：**定理或方法的呈现必须遵循认知的自然顺序，而非教科书式的"定义→定理→证明"机械排列**。具体而言：

- 如果定理本身不依赖复杂前置知识（如鸽巢原理），**从具体问题入手**——先让学生感受"碰到这类问题该怎么办"，在解题过程中自然而然引出定理方法
- 如果定理的证明过程本身具有结构之美（如柯西-施瓦茨不等式、二次互反律），**以证明过程为引入**——每一种证法都是一次"为什么要这样"的回答，多把钥匙、多条路径，让学生在对比中把握定理的深层结构

模板只是参考框架，不是紧箍咒。不同专题需要不同的叙事弧线，唯一不变的是：**让学生在读完文章后，不仅知道"怎么做"，更理解"为什么可以这么做"**。只有理解深层结构，学生才能真正内化，而非机械背诵方法。

## 文章分类与结构

文章分为三大类，各类的结构范式以典范文章为基准，实际写作时根据专题特性和写作宗旨灵活调整。

### 第一类：理论专题文

给定一个专题方向，系统展开。分为两个子类：

#### 1a. 某定理及其运用

围绕一个核心定理，讲授其证明、变形与应用。

**典范文章**：No1 鸽巢原理、No7 柯西-施瓦茨不等式、No8 母函数、No9 二次剩余

**两种叙事路径**（根据定理特性二选一或融合）：

| 路径 | 适用场景 | 结构骨架 | 代表 |
|------|---------|---------|------|
| 问题驱动 | 定理本身易懂，不依赖前置知识 | 破题分析（抛出具体问题）→ 解法详录（解题中引出定理）→ 启发与一般化 → 教学后记 | 鸽巢原理 |
| 证明驱动 | 定理的多种证法各有"何以所以然"的启示 | 引入（铺垫直觉）→ N 种证法（每法提炼核心思想 + 直观度评级）→ 破题实例（逐级递进）→ 启发与一般化 → 教学后记 | 柯西不等式、二次剩余 |

**共同要求**：
- 开篇以名言引语起笔，随后以自然方式进入主题——禁止突兀摆定理
- 理论专题文必须在引入部分融入数学史（300~500 字），按四个维度选材：概念起源 / 现实驱动 / 迭代完善 / 趣味故事。以"一个想法走了多少年"的叙事线索串联，避免教科书式年代列表。数学史内容应与后文的技术内容形成呼应
- 实例逐级递进（高考 → 竞赛 → 大学低年级），每道题带"识别信号"——看到什么结构就该想到这个工具
- "启发与一般化"必须提炼通用思维（对称性、拆解转化、限制催生创造力等），可埋后续伏笔

#### 1b. 某类问题的一般方法

围绕一类问题的通用方法论或思考框架展开。

**典范文章**：No4 组合极值、No3 网格作图

**典型结构**：

```
破题分析（定义问题类型 + 提炼通用框架）
  ↓
经典案例 / 模型（3~4 个，由浅入深）
  ↓
启发与一般化（不同案例的共同基因 + 高阶推广）
  ↓
教学研究后记
```

**关键要点**：
- 开篇先定义"这类问题长什么样"，建立识别直觉
- 给出通用方法论框架，再以案例逐一验证和深化——先给地图，再走景点
- 案例要有多样性，同一方法在不同场景的变形要体现
- 同样需要数学史穿插（该方法如何产生、如何迭代）

### 第二类：典型例题剖析文

围绕一道典型题目，以多种方法深挖，展示数学结构的丰富性。

**典范文章**：一道恒等式三把钥匙（Σ C(n,k)² = C(2n,n)）

**典型结构**：

```
题目呈现
  ↓
破题分析（目标导向 + 条件导向 + 解题策略概述）
  ↓
多法详录（每种方法一套完整推导，互相形成对话）
  ↓
方法归一（回头看：不同方法的共同本质是什么？）
  ↓
启发与一般化（推广到更一般的定理或概率视角等）
  ↓
教学研究后记（一题多解的教学价值）
```

**关键要点**：
- 方法之间不是罗列关系，而是**对话关系**——方法二回应方法一的不足，方法三揭示方法一和二的共同结构
- "方法归一"是灵魂：做完所有方法后，必须有一段总结不同方法的底层一致性
- 每法提炼识别信号："看到 XX 结构 → 想到 YY 方法"
- 破题分析双线并行：目标导向（从结论反推需要什么）+ 条件导向（从已知正向挖掘隐藏结构）

### 第三类：习题总解

对往期多篇文章的课后习题做系统性汇总解答，将散落在各篇的练习串联成一个有机整体。一题多解处标注"另解"，篇末不留思考题（习题总解本身即是对此前所有思考题的回应）。

**典范文章**：No10 前九篇课后习题全解

**总体结构**（三部分有机咬合，通过交叉引用互连）：

```
序言（说明三部分定位与使用方式）
  ↓
Part 1：习题集（独立可打印的习题册）
  ↓
Part 2：习题解答（逐篇逐题三段式）
  ↓
Part 3：方法地图（跨文章方法簇串讲）
  ↓
后记（升华点题，回应序言）
  ↓
预告
```

**Part 1 习题集**：

将全部习题按难度分为三档排列，每题标注方法标签和解题提示：

| 难度档 | 定位 | 特征 |
|--------|------|------|
| 基础巩固 | 检验基本概念和定理的掌握 | 直接应用，无需技巧 |
| 能力提升 | 需要对定理有一定理解后灵活运用 | 需要组合多个知识点 |
| 思维拓展 | 需要深度理解和创造性思维 | 可能有多种解法，解法之间互相补充 |

- 每题格式：`编号. [文章来源] 题目描述` + 方法标签（如 `鸽巢原理` `双计数`）+ 一行解题提示
- 每题末尾附交叉引用：`解答见 §2.X / 方法归类见 §3.Y`
- Part 1 可独立打印为习题册，不依赖后文即可使用

**Part 2 习题解答**：

按文章时间顺序（No1 → No2 → …）逐篇逐题展开，每题三段式：

```
破题分析（1~2 句直击关键难点与方法选择理由）
  ↓
标准求解（完整推导过程，一题多解处标注"另解"）
  ↓
原文呼应（回链原文具体段落，建立知识锚点）
```

**关键要点**：
- 解答顺序按文章时间线，而非难度——读者可以随文章进度自然查阅
- "原文呼应"必须指向原文具体位置（如"见 No3 §2 网格作图的构造方法"），让读者回到原文巩固理解
- 同一题有多个解法时，每种解法完整呈现，标注"另解一""另解二"，各解法之间形成互补而非罗列

**Part 3 方法地图**：

将所有习题按**方法簇**重新归类，跨文章串讲。每个方法簇包含：

- **对比表格**：列出所有属于该簇的题目，标注题目来源、所用工具、关键技巧
- **串讲段落**：揭示同一方法在不同题目中的变形和联系，提炼通用思维

**典型方法簇分类**（7 簇示例，实际写作时按习题实际方法分布调整）：

| 方法簇 | 涵盖主题 |
|--------|---------|
| 鸽巢原理与双计数 | 鸽巢原理、双计数技巧 |
| 母函数与生成函数 | 母函数构造、生成函数运算 |
| 不等式技巧 | 柯西不等式、均值不等式、排序不等式 |
| 数论符号与模运算 | 同余、二次剩余、数论函数 |
| 连分式与逼近 | 连分式展开、有理逼近 |
| 几何构造与格点 | 几何作图、格点问题 |
| 组合模型与概率视角 | 组合恒等式的概率解释、组合极值 |

**关键要点**：
- 方法簇的数量和名称根据实际习题内容生成，不强制 7 簇
- 每簇必须先表后文：表格给读者快速定位自己关心的题目，串讲揭示方法的深层一致性
- 串讲务必回链 Part 2 的具体解答位置，形成"总—分—总"的阅读闭环

**习题总解独有的约束**：
- 序言必须明确三部分的定位关系：Part 1 独立习题册 → Part 2 详细解答 → Part 3 俯视全局
- 后记必须回应序言中的名言引语或核心隐喻，形成首尾呼应
- 篇末预告下一期主题，**不留思考题**——习题总解本身就是对往期所有思考题的集中回应

### 通用要素（三类文章共有）

无论哪种类型，每篇文章末尾必须包含：

1. **思考题**：1~3 道与本文主题相关的思考题
2. **预告**：预告下一期主题，形成连载感。格式如"下一期我们将讨论……"
3. **标题格式**：`NoX. 主题关键词` 或直接以主题命名

## 语言规范

- **中英混排**：数学术语、定理名、人名用英文（Pascal定理、the proof is trivial、Grübler公式），论述用中文。
- **第一人称复数**："我们"贯穿全文，建立与读者的共研关系。避免使用"笔者"（仅在后记中偶尔可用）。
- **学术 + 口语混合**：主体保持学术严谨，但允许自然的口语流露，如"等等，这不是导数的几何意义吗？""读者不妨自己证明"。
- **禁止使用任何表情符号**。
- **引用格式**：名言引用使用中英双语（原文 + 中文翻译），标注出处。

## 内容轮换规则

代数、几何、组合、数论四大板块交替。连续两篇不写同一板块。优先选择与近期文章形成呼应的主题（如上一篇写了单尺作图，下一篇可写单规作图或网格作图）。

## 写作流程

写作流程严格分为两阶段：头脑风暴 → 列提纲与写作。两阶段的分工不可混淆。

### Phase 1：头脑风暴（只出内容，不碰结构）

**目的**：确定文章的行文思路和内容要点，不涉及章节命名和结构编排。

**产出物**：
- 中心思想（一句话概括本文要传递的核心洞察）
- 叙事主线（从哪开始、经过哪些关键节点、在哪收束）
- 覆盖的知识点和案例清单
- 各知识点的侧重点和行文重点
- 数学史素材（来源、与正文的呼应点）

**禁止事项**：在此阶段设计章节结构、命名小节、编排序号。这些属于 Phase 2。

### Phase 2：列提纲与写作（严格对齐模板）

1. **确定主题与板块**（代数/几何/组合/数论），遵循内容轮换规则
2. **确定文章类型**：理论专题文（1a 定理运用 / 1b 一般方法）、典型例题剖析文或习题总解
3. **对照模板列提纲**：打开本 skill 中对应类型的典范文章模板，将 Phase 1 的内容素材填入模板的结构槽位。**提纲必须保持与模板一致的结构骨架**（序言→破题分析→解法详录→启发与一般化→教学后记→思考题→预告），确保每篇文章的格式特征统一，作为公众号的标识之一。模板是框架约束，专题特性可在框架内灵活调整细节，但不可改变骨架顺序和模块边界。
4. **按提纲写作**：逐节撰写正文，每完成一节即检查是否偏离提纲
5. **必须使用 write_file 将文章保存为 Markdown 文件到 output 目录**。文章内容仅通过 yyb-product 卡片在右侧预览区展示，**绝对禁止在聊天对话框中输出文章全文**
6. **检查**：写作宗旨是否贯穿（读者能否理解"何以所以然"）、中英混排是否自然、是否有思考题和预告、是否呼应精神纲领
